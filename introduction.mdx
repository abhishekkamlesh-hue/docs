# VoiceGenie Dialer Service - API Documentation

Welcome to the VoiceGenie Dialer Service API documentation. This guide will help you integrate with our AI-powered voice campaign management system.

## Getting Started

To get started with the VoiceGenie API, you'll need:

1. An API key (x-api-key header)
2. Your base URL endpoint
3. Review the authentication section

## Base URL

```
https://vb-seamless-dev-backend.oriserve.com/
```

## Authentication

All API requests require authentication using an API key in the request headers:

```bash
x-api-key: your-api-key-here
```

## API Features

VoiceGenie provides a comprehensive suite of APIs to manage your voice campaigns:

### **Campaign Management**
Create, manage, and monitor your voice campaigns with full lifecycle control.
- [Update Campaign](/api-reference/campaigns/update-campaign-data)

## Rate Limiting

API requests are rate-limited to ensure service stability. Default limits apply per endpoint.
