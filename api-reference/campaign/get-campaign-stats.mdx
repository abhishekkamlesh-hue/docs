## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="Campaign Statistics">
    <ResponseField name="campaign_id" type="string">
      Unique identifier of the campaign
    </ResponseField>
    
    <ResponseField name="campaign_name" type="string">
      Name of the campaign
    </ResponseField>
    
    <ResponseField name="total_contacts" type="integer">
      Total number of contacts in the campaign
    </ResponseField>
    
    <ResponseField name="calls_completed" type="integer">
      Number of successfully completed calls
    </ResponseField>
    
    <ResponseField name="calls_failed" type="integer">
      Number of failed calls
    </ResponseField>
    
    <ResponseField name="calls_in_progress" type="integer">
      Number of calls currently in progress
    </ResponseField>
    
    <ResponseField name="calls_pending" type="integer">
      Number of calls yet to be initiated
    </ResponseField>
    
    <ResponseField name="success_rate" type="number">
      Percentage of successful calls (0-100)
    </ResponseField>
    
    <ResponseField name="average_call_duration" type="integer">
      Average call duration in seconds
    </ResponseField>
    
    <ResponseField name="total_call_duration" type="integer">
      Total duration of all calls in seconds
    </ResponseField>
    
    <ResponseField name="call_status_breakdown" type="object">
      <Expandable title="Status Breakdown">

          Recipient was busy
        </ResponseField>
        
        <ResponseField name="no_answer" type="integer">
          No answer from recipient
        </ResponseField>
        
        <ResponseField name="failed" type="integer">
          Failed due to technical issues
        </ResponseField>
        
        <ResponseField name="voicemail" type="integer">
          Reached voicemail
        </ResponseField>
      </Expandable>
    </ResponseField>
    
    <ResponseField name="started_at" type="string">
      ISO 8601 timestamp when campaign started
    </ResponseField>
    
    <ResponseField name="completed_at" type="string">
      ISO 8601 timestamp when campaign completed (null if still running)
    </ResponseField>
  </Expandable>
</ResponseField>

## Performance Notes

- **Response Time**: Typically < 500ms for campaigns with < 10,000 contacts
- **Heavy Load**: Campaigns with 50,000+ contacts may take 2-3 seconds
- **Real-time Data**: Statistics are updated in real-time as calls complete
- **Caching**: Stats are cached for 30 seconds to improve performance
- **Rate Limiting**: 200 requests per minute per campaign

## Use Cases

- **Dashboard Display**: Show real-time campaign performance
- **Monitoring**: Track campaign progress programmatically
- **Reporting**: Generate custom reports and analytics
- **Alerting**: Set up alerts based on success rates or completion status
