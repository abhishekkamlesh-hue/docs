## Overview

Search for available phone numbers by country, number type, pattern matching, and other criteria.

---

##  Authentication

<ParamField header="Cookie" type="string" required>
  Your API key for authentication
</ParamField>

<ParamField header="Accept" type="string">
  application/json, text/plain, */*
</ParamField>

<ParamField header="Origin" type="string">
  Origin of the request (e.g., https://vb-seamless-dev-backend.oriserve.com/)
</ParamField>

<ParamField header="User-Agent" type="string">
  Browser or client user agent
</ParamField>

---

##  Query Parameters

<ParamField query="country_iso" type="string" required>
  Two-letter ISO country code
  ```
  IN
  ```
  
  **Examples:** `US`, `GB`, `IN`, `AU`, `CA`
</ParamField>

<ParamField query="type" type="string">
  Number type
  ```
  local
  ```
  
  **Options:**
  - `local` - Local phone numbers
  - `tollfree` - Toll-free numbers
  - `mobile` - Mobile numbers
</ParamField>

<ParamField query="pattern" type="string">
  Pattern to match in the phone number
  ```
  1234
  ```
  
  Use wildcards: `123*` or `*456`
</ParamField>

<ParamField query="region" type="string">
  Specific region/state code
  ```
  Maharashtra
  ```
</ParamField>

<ParamField query="services" type="string">
  Comma-separated service capabilities
  ```
  voice,sms
  ```
  
  **Options:** `voice`, `sms`, `mms`
</ParamField>

<ParamField query="limit" type="integer">
  Maximum results to return
  ```
  20
  ```
  
  **Default:** 20, **Max:** 100
</ParamField>

## Code Examples

### cURL:
```bash
curl -X GET "https://vb-seamless-dev-backend.oriserve.com/search?country_iso=IN&type=local&limit=20" \
  -H "x-api-key: YOUR_API_KEY"
```

### JavaScript:
```javascript
const axios = require('axios');

async function searchPhoneNumbers(params) {
  try {
    const response = await axios.get(
      'https://vb-seamless-dev-backend.oriserve.com/search',
      {
        params: {
          country_iso: 'IN',
          type: 'local',
          pattern: '1234',
          limit: 20
        },
        headers: {
          'x-api-key': 'YOUR_API_KEY'
        }
      }
    );
    
    console.log('Available numbers:', response.data.data.objects);
  } catch (error) {
    console.error('Error:', error.response.data);
  }
}

searchPhoneNumbers();
```

### Python:
```python
import requests

def search_phone_numbers():
    url = "https://vb-seamless-dev-backend.oriserve.com/search"
    
    params = {
        "country_iso": "IN",
        "type": "local",
        "pattern": "1234",
        "limit": 20
    }
    
    headers = {
        "x-api-key": "YOUR_API_KEY"
    }
    
    response = requests.get(url, params=params, headers=headers)
    
    if response.status_code == 200:
        data = response.json()['data']
        print(f"Found {data['meta']['total_count']} numbers")
        for num in data['objects']:
            print(f"- {num['number']} (â‚¹{num['monthly_rental_rate']}/month)")

        print("Error:", response.json())
```

##  Pricing Information

Each number in the response includes:
- `monthly_rental_rate` - Monthly cost (USD)
- `setup_rate` - One-time setup fee
- `voice_rate` - Per-minute voice call cost
- `sms_rate` - Per SMS cost
