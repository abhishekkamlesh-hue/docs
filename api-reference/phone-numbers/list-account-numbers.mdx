##  Overview

Retrieve all phone numbers from your account with populated script details (agentIds), inbound call configuration, and metadata. Supports pagination and filtering by country and deletion status.

---

##  Query Parameters

<ParamField query="pageSize" type="integer" default="1000">
  Number of results per page. **Max:** 1000
</ParamField>

<ParamField query="page" type="integer" default="1">
  Page number (1-indexed)
</ParamField>

<ParamField query="country_iso" type="string">
  Filter by country ISO code (e.g., `IN`, `US`, `GB`)
</ParamField>

<ParamField query="isDeleted" type="string">
  Set to `true` to include only deleted numbers. Defaults to showing non-deleted numbers.
</ParamField>

## Field Descriptions

### PhoneNumber Object Fields

| Field | Type | Description |
|-------|------|-------------|
| `_id` | string | MongoDB ObjectId of the phone number document |
| `number` | string | Phone number (E.164 format without +) |
| `status` | string | Current status (e.g., `active`, `unrented`) |
| `dialer` | string | Dialer provider (e.g., `plivo`) |
| `agentIds` | array | Array of Script ObjectIds associated with this number |
| `associatedScripts` | array | Populated script objects with `_id` and `botName` |
| `configuredAgentIdForInboundCalls` | object/null | Script configured for inbound calls (populated with `_id` and `botName`) |
| `isInboundCallEnabled` | boolean | Whether inbound calls are configured (derived field) |
| `metadata` | object | Additional metadata (e.g., `alias`) |
| `createdAt` | string (ISO 8601) | Creation timestamp |
| `updatedAt` | string (ISO 8601) | Last update timestamp |

### Meta Object Fields

| Field | Type | Description |
|-------|------|-------------|
| `page` | number | Current page number |
| `pageSize` | number | Number of results per page |
| `total` | number | Total count of matching numbers |
| `totalPages` | number | Total number of pages |

---

## Notes

- **Population:** The `agentIds` field is populated with script details (`_id` and `botName`)
- **Derived Field:** `isInboundCallEnabled` is computed based on whether `configuredAgentIdForInboundCalls` is set
- **Associated Scripts:** Returned as a formatted array with `_id` and `botName` for easy display
- **Pagination:** Use `page` and `pageSize` to navigate through large result sets
- **Filtering:** Combine `country_iso` and `isDeleted` query params for targeted filtering
- **Default Sort:** Numbers are sorted by `number` field in descending order
- **isDeleted Logic:** By default, shows non-deleted numbers; set `isDeleted=true` to see only deleted ones
- **Max Page Size:** Maximum `pageSize` is 1000 to prevent performance issues
