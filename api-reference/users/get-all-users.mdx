##  Overview

Retrieve all users in the system. Passwords are automatically excluded from the response for security.

## Code Examples

### cURL:
```bash
curl -X GET "https://vb-seamless-dev-backend.oriserve.com/getAllUsers" \
  -H "x-api-key: YOUR_API_KEY"
```

### JavaScript:
```javascript
const axios = require('axios');

async function getAllUsers() {
  try {
    const response = await axios.get(
      'https://vb-seamless-dev-backend.oriserve.com/getAllUsers',
      {
        headers: {
          'x-api-key': 'YOUR_API_KEY'
        }
      }
    );
    
    console.log('Total users:', response.data.data.length);
    console.log('Users:', response.data.data);
  } catch (error) {
    console.error('Error:', error.response.data);
  }
}

getAllUsers();
```

### Python:
```python
import requests

def get_all_users():
    url = "https://vb-seamless-dev-backend.oriserve.com/getAllUsers"
    
    headers = {
        "x-api-key": "YOUR_API_KEY"
    }
    
    response = requests.get(url, headers=headers)
    
    if response.status_code == 200:
        data = response.json()
        print(f"Total users: {len(data['data'])}")
        for user in data['data']:
            print(f"- {user['name']} ({user['email']}) - {user['accessLevel']}")
    else:
        print("Error:", response.json())

get_all_users()
```

## Use Cases

### 1. User Management Dashboard:
```javascript
async function loadUsersDashboard() {
  const response = await axios.get('/getAllUsers');
  const users = response.data.data;
  
  const stats = {
    total: users.length,
    active: users.filter(u => u.active).length,
    blocked: users.filter(u => u.isUserIdBlocked).length,
    admins: users.filter(u => u.accessLevel === 'admin').length
  };
  
  console.log('User Statistics:', stats);
}
```

### 2. Filter Active Users Only:
```javascript
async function getActiveUsers() {
  const response = await axios.get('/getAllUsers');
  const activeUsers = response.data.data.filter(user => 
    user.active && !user.isUserIdBlocked
  );
  
  return activeUsers;
}
```

### 3. Check User Permissions:
```javascript
function canUserAccessRoute(user, route) {
  return user.allowedRoutes.includes(route) || 
         user.defaultRouteNames.includes(route);
}
```
